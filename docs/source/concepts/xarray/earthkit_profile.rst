Profiles: earthkit
-------------------------

This is the definition of the built-in ``"earthkit"`` profile for the :ref:`Xarray engine <xr_engine>`. This is the default profile used when no profile is specified. It is using the format independent metadata keys from :py:class:`~earthkit.data.core.field.Field` to generate the Xarray dataset.
